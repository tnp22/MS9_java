3계층 패턴

프리젠테이션 계층
ㄴ사용자와 직접 상호작용하는 계층으로, UI역할을 한다.

비즈니스 로직 계층
ㄴ핵심 로직을 처리하는 계층
 - 데이터 처리 로직
 - 업무 프로세스(비즈니스 규칙)로직
 
데이터 계층
ㄴ데이터와 상호작용하는 계층
 -데이터 계층 구성요소
 	- DTO (Data Transfer Object)
 		데이터 전송 객체
 		 - 서로다른 클래스 간에, 서로 다른 계층 사이에서 데이터를 전송하기 위해 사용
 		 - 주로, 데이터베이스로부터 읽어온 데이터 단위를 객체로 가져와서 사용
 		 - DB(테이블) ----> 자바 (객체)
 		 - ex) 게시판 정보 클래스 Board
 	- DAO (Data Access Object)
 		데이터 접근 객체
		 - 데이터베이스에 접근을 하여 상호작용을 하는 객체
		 - 업무 로직과 데이터베이스 연동 로직을 분리하기 위해서 사용
		 - ex) 게시글 데이터 접근 클래스 BoardAccess
